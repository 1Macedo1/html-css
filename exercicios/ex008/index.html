Principais formatações:

Negrito / Destaque

Nesta frase temos um termo em negrito, usando a tag <b>. (Sem semântica)

Nesta frase, temos um termo em destaque, usando a tag <strong>(Com semântica)

Itálico / Ênfase

Nesta frase temos um termo em itálico, usando a tag <i/>. Sem semântica.

Nesta frase, temos um termo em ênfase usando a tag <em/>. Com semântica.

Texto Marcado

Podemos criar também um texto marcado, usando a tag <mark>. No caso a formatação style depois de mark fa parte do css

Aqui nesse outro exemplo, está marcado só pelo html sem o estilo do css. Dá também para marcar o style lá no início como se fosse uma variável, logo abaixo da tag title. Daí onde tiver o mark no html, vai assumir a cor do style que vc informar.

Texto grande e pequeno.

Estamos criando um texto grande e um texto pequeno neste parágrafo, usando as tags <big/> e <small/>

Texto excluído

Podemos marcar um texto como exluído para indicar que ele deve ser lido, mas não considerado. Usando a tag <del/>

Texto inserido

podemos marcar um texto como inserido para dar ênfase e indicar que ele foi adicionado depois. Usando a tag <ins/>

Texto sobreescrito.

Para inserir coisas do tipo x20+3. Usei a tag <sup>

Texto subscrito

Para inserir coisas do tipo h2o(água). Usei a tag <sub>
