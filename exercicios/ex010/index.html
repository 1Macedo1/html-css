Usando links externos

Para criar um link, vc vai usar a tag <a href=""> A letra a é de âncora e o href é de referência do hyperlink


Você pode acessar o meu repositório público no GitHub

Note que no link acima, o site do link sobrepõe o seu site e para isso não acontecer, basta vc colocar target="_blank" rel="external " após o endereço do link.. ficando assim: <a href="https://www.youtube.com/channel/UCrWvhVmt0Qac3HgsjQK62FQ" target="_blank" rel="external">

Ou acesse o canal do Curso em Vídeo pelo youtube..
