Já no Estilo Externo, fica mais prático pq vc não tem que ficar copiando os vários estilos para as várias páginas, basta vc criar um link : css e criar o arquivo.css , feito isso vc copia todo aqle estilo, ou cria novos estilos e insere somente a linha do link nas páginas.

Dessa forma, faz com que todas as páginas tenham o mesmo stilo que puxa atravé do link q vc criou.

IMPORTANTE Não esquecer de colocar no início do arquivo css a regra @charset "UTF-8"; que serve para acentuações, tem que colocar!
