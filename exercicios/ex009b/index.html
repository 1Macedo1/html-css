Listas ordenadas

Para isso, precisaremos de duas tags a <ol/> e dentro dela a <li/>.

  1. Acordar
  2. agradecer a Deus pela vida
  3. Olhar o celular
  4. Estudar

Dessa forma vai ordenando como uma lista.

Dá para alterar o tipo de ordenamento também, ao invés de colocar a ordem 1,2,3,4.. pode ser A,B,C.. ou outros 7 tipos que o html re sugere, basta colocat a tag <type> dentro da tag <;ol> E informar o tipo que você quer

Listas Não-Ordenadas

Para isso, precisaremos de duas tags a <ul/> e dentro dela a <li/>.

  • arroz
  • feijão
  • carne
  • frango
  • açucar
  • limão

    • Dá para alterar o tipo de ordenamento também, ao invés de colocar a ordem 1,2,3,4.. pode ser A,B,C.. ou outros 7 tipos que o html re sugere, basta colocat a tag <type> dentro da tag <ol> E informar o tipo que você quer, pode ser, disc, circle, square.

    Ficará assim: <ul type="circle">

Lista de definições

Para criar uma lista de definição você habilita as tags <dl/> e depois vc habilita as tags: <dt>para um termo e <dd> para uma definição.

    <dl>
        <dt>   <dt/>
        <dd>   <dd/>
    <dl/>
